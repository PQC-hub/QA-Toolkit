Features

System Audit

Analyze CPU, RAM, disk usage and Windows system information.

Temp Cleanup

Remove temporary files, cache and leftover update files.

Startup Manager

View and disable unnecessary startup programs.

System Health Score

Calculate overall system health score (0-100).

HTML Report

Generate professional system reports in HTML format.

Backup & Restore

Backup system data and restore when needed.

Application Interface

  • Administrator privileges

powershell -ExecutionPolicy Bypass -File QA-Toolkit.ps1
